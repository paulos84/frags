FuncMols

This site provides users with the ability to access and store various information relating to individual functional molecules.
Is intended for end users and suppliers of registered or known organic compounds. Such users may be from either industry,
academia or the general public.

Basic usage

Compounds are classified into three types: fragrance and flavor, medicinal and pharmaceutical, functional food (nutraceutical).
Using the search form on the sidebar, exact compounds may be looked up using their identifiers (either CAS number or InChiKey),
or filtered lists matching the name lookups obtained.

If a compound does not exist within the database, the relevant form can be used to add it to the database. The form differs
slightly between fragrance compounds and other bioactives, since fragrance compounds are identified through their registered
CAS numbers and require the submission of sensory characteristics. Upon entering a valid chemical identifier, the page will be
updated with the chemical name and structure of the compounds to be added, in order to confirm it is the expected compound.
If a compound matching the chemical identifier already exists in the database, a link to its page will be provided.

Each compound has a page through which various information can be obtained and through which the user can save arbitrary data
relating to the compound. The page also allows visitors to access listed commercial sources and a list of links to recently-
published scientific literature. Through selecting checkboxes next to the listed items, these can be saved to a users profile
for future reference. A form is available to enable commercial sources to be listed and thereby accessed by other site users.

To access all compound information stored on a user profile, a link on the drop-down menu on the top navigation bar should be used.
This page provides a list of compounds with a summary of the stored data, along with links to view and update it.

Privacy policy

Email addresses for user registration will be kept confidential and not shared with third parties. Data stored within user profiles
are kept private and the site implements a robust level security. In order to prevent brute force login attacks, accounts will be
locked for a period of time upon a successive number of failed login attempts. A password reset link is provided on the login page.
